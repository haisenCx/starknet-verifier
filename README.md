## Installation

1. Clone this repo:

```console
git clone git@github.com:reddio-com/starknet-verifier.git
```

2. Install NPM packages:

```console
cd starknet-verifier
npm install
```

3. .sepolia.env

```console
export HARDHAT_ENV=sepolia
export RPC_PROVIDER=...
export ADMIN_PRIVATE_KEY=...
export ETHERSCAN_KEY=...
```


4. Deploy

```console
npm run deploy:sepolia
```
